# Fellah Landing

Marketing site for [fellah.tn](https://fellah.tn) — the agri-tech platform for Tunisian farmers.

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Deployed on Vercel → `fellah.tn`

## Development

```bash
npm install
npm run dev   # → http://localhost:3001
```
